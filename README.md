
# mcmod-info-mirror

为各平台的 Mod 信息缓存加速

预计不会提供**文件**下载，成本过高

基于 [Curseforge](https://curseforge.com/) 和 [Modrinth](https://modrinth.com/) 作为信息源

参考 [CFCore](https://docs.curseforge.com/) 和 [Modrinth Docs](https://docs.modrinth.com/)
